# DjangoAir ERP

---

## This project implements airline management, creating flights, selling tickets (which is done with WayForPay) and sending them to the user in PDF format, creating staff (distributing permits).

### Example:

*You can clone and run the project with the "docker-compose -f docker-compose.prod.yml up --build" command* 



**Don't forget to create an .env.dev file and fill it with the appropriate data**
